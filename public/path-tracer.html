Path Tracer (work in progress)

Individual Project

Software/Tools:

Description:
Basics of a photorealistic path tracer developed using C++.
This path tracer contains:
---- Positionable cameras with depth of field (bokeh)
---- Materials such as Lambert, Metal, Dielectrics
---- Reflection and Refraction
---- Light sources

This is a work in progress so please check back for more updates/features. Upcoming features include:
---- OBJ loader
---- Motion Blur
---- Cornell Box
---- Surface Textures
---- Solid Textures such as Perlin Noise
---- Parallel processing
---- Volume rendering, etc.

Results:
A simple scene with 3 spheres having different material properties - Lambert, Metal, Fully Reflective


Procedural checker pattern for the ground plane, lambert shaded sphere, reflective sphere, and two spherical light sources. There is some sort of distortion present (seen below light spheres) and a fix should be on its way (soon, hopefully!)
