Developer Reel

1. Arthur - Real time OpenGL rendering

Developed using C++, OpenGL and GLSL. The UI is created using ImGUI (https://github.com/ocornut/imgui). ASSIMP and SOIL libraries have been used for model loading and image loading respectively. Implemented Physically based rendering, deferred rendering and forward rendering. Features include PBR textures, image based lighting using HDR maps, screen space ambient occlusion, point and directional lights, etc.

2. Flocking simulation

Developed using Processing. Based on Craig Reynolds 1987 SIGGRAPH paper - Flocks, Herds, and Schools: A Distributed Behavioral Model. Includes flock centering, velocity matching, collision detection and multiple flocks.

3. Path tracer (work in progress)

Developed using C++. Monte Carlo path tracer that includes materials, textures, reflection, refraction, light sources, etc.
