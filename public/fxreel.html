FX Reel

1. The Novice - Pyro

Created pyro effects using Houdini. A tornado setup was developed to move in accordance with the characters movement. This involved extracting the character animation, creating a curve, and building the volume based on the curve animation. The character velocity was used in the simulation to create wisps of smoke from the tornado. This method was also extended to work on a custom curve created by the user.

2. The Novice - Glow

Glow effects were done using Houdini by creating the glow pattern on the T-pose and then transferring them on to the animated character using rest positions. This ensured the glow would progress uniformly over the characters surface. Not included in final film.

3. Christmas card

Worked on a Christmas-based short story along with other interns during my internship at Framestore, NY. All FX work done using Houdini. Created the falling snow by modifying an internal HDA. Skating trails and spray was created using points on the feet of the charaters as independent POP sources.

4. Knot today - Summer industry course

Created steam effects using Houdini as part of a 6 person team that worked on a 30 second animated short under the guidance of supervisors from Walt Disney Animation Studios. Steam dispersion handled using POP velocities in a pyro simulation.

5. Lava simulation (work in progress)

Created using Houdini. Flip fluids with custom variable viscosity and temperature control.
